# 贡献指南

欢迎一起完善 Eve 中文文档与工程实践指南。

## 写作目标

本项目不是逐字翻译站点，而是面向中文开发者解释 Eve 的设计、用法和工程落地方式。

每篇文档建议包含：

1. 这个概念解决什么问题；
2. 官方文档的核心含义；
3. 白话解释；
4. 最小代码示例；
5. 本地部署或工程实践建议；
6. 常见误区。

## 风格约定

- 默认使用简体中文。
- 不要大段复制官方文档原文。
- 保留上游链接，方便读者核对最新行为。
- 明确区分“官方说明”和“个人/项目建议”。
- 对 beta 功能要标注不稳定风险。

## 目录约定

- `docs/core-concepts/`：核心概念解释。
- `docs/deployment/`：部署、自托管、运行时配置。
- `docs/guides/`：官方 guide 的中文解读和实践。
- `docs/examples/`：面向实际项目的架构和示例。
- `examples/`：可运行示例项目。

## 校对清单

提交前请检查：

- 是否保留官方链接；
- 是否避免复制大段原文；
- 是否说明适用场景和不适用场景；
- 是否有代码块语言标识；
- 是否有本地部署建议；
- 是否标注 beta / 可能变化的内容。
